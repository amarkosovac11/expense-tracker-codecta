# EXPENSE TRACKER

A simple Expense Tracker app for managinig: users, transactions, categories and saving goals.

## Technologies used

Backend:  ASP.NET Core (.NET), EF Core, JWT Auth and Swagger
Frontend: React (Vite)
Database: PostgreSQL

## Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/amarkosovac11/expense-tracker-codecta.git
cd expense-tracker-codecta
```

### 2️⃣ Run the application

```bash
docker compose up --build
```

### 3️⃣ Open the application

Backend (Swagger): http://localhost:8080/swagger

Frontend: http://localhost:5173
